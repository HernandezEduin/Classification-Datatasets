# Classification Datatasets
 Simple example on how to train models on the classification tasks using pytorch.

## Available Datasets for Training and Validation:
1. MNIST
2. KMNIST
3. FashionMNIST
4. CIFAR10
5. CIFAR100

## Available Models:
1. Linear Regression
2. Multi Layer Perceptron (MLP)
3. LeNet
4. ResNet
5. VGG
